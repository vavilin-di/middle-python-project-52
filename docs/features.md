# Возможности

## Управление пользователями

- Регистрация нового пользователя
- Аутентификация (вход/выход) с flash-сообщениями об успехе
- Редактирование собственного профиля
- Удаление собственного профиля
- Просмотр списка всех пользователей

## Управление статусами

- Создание нового статуса задачи
- Редактирование существующего статуса
- Удаление статуса
- Просмотр списка всех статусов

## Управление метками

- Создание новой метки
- Редактирование существующей метки
- Удаление метки
- Просмотр списка всех меток

## Управление задачами

- Создание задачи с указанием названия, описания, статуса, исполнителя и набора меток
- Просмотр детальной информации о задаче
- Редактирование задачи
- Удаление задачи
- Просмотр списка всех задач с возможностью фильтрации

## Фильтрация задач

- По статусу
- По исполнителю
- По меткам
- Только задачи, где текущий пользователь является автором

## Интернационализация (i18n)

- Поддержка русского и английского языков
- Язык по умолчанию — русский
- Переключение языка через URL-параметр (`/i18n/setlang/`)
- Осмысленные строковые идентификаторы в `msgid` (например, `TaskVerboseName` вместо `"Задача"`)
- Автоматическое определение языка через `LocaleMiddleware`
- Подробнее: [i18n.md](i18n.md)

## UI/UX

- Адаптивный интерфейс на Bootstrap 5
- Иконки Bootstrap Icons
- Flash-сообщения об успешных/ошибочных операциях
- Единый шаблон для списковых представлений
