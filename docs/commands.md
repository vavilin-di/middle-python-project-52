# Команды Makefile

| Команда              | Описание                                       |
| -------------------- | ---------------------------------------------- |
| `make install`       | Установить зависимости через `uv sync`         |
| `make build`         | Полная сборка (установка + статика + миграции) |
| `make migrate`       | Применить миграции базы данных                 |
| `make dev`           | Запустить сервер разработки                    |
| `make test`          | Запустить тесты                                |
| `make start`         | Запустить production-сервер (Uvicorn)          |
| `make collectstatic` | Собрать статические файлы                      |
| `make render-start`  | Запустить Uvicorn для Render.com               |

## Переменные

- `PORT` — порт для сервера (по умолчанию `8000`)

Пример:

```bash
make dev PORT=8080
